# 날마다성경 웹앱  
> 52주 말씀 암송 · 365일 성경 통독

주평강교회 성도들을 위한 **모바일 친화 웹앱(PWA)** 입니다.  
말씀 암송과 성경 통독을 **간단하고 지속 가능하게** 돕기 위해 제작되었습니다.

🔗 https://nalmada.netlify.app

---

## ✨ 주요 기능

### 📌 52주 말씀 암송
- 주차별 암송 구절 제공
- 이전 / 이번 / 다음 주 빠른 이동
- 개인 진도 자동 저장 (기기별 localStorage)
- 링크 공유 가능

### 📖 365일 성경 통독
- 하루 분량 자동 계산
- 완료 체크 및 누적 진행률 표시
- 오늘 / 이전 / 다음 분량 이동
- 자동 다음 분량 옵션 제공

---

## 📱 PWA 지원
- 홈 화면에 설치 가능 (Android / iOS)
- 앱처럼 전체화면 사용
- 오프라인 캐시 지원 (Service Worker)


---

## 🛠 기술 스택
- HTML / CSS (Tailwind CSS)
- JavaScript (Vanilla)
- jQuery (경량 DOM 처리)
- PWA (Manifest + Service Worker)
- Netlify Hosting

---

## 🎯 목적
> “말씀으로 다시 돌아가는 교회”

이 웹앱은 **주평강교회 성도들의 말씀 생활을 돕기 위한 도구**로,  
광고·수익·추적 없이 오직 **편의성과 지속성**에 초점을 두고 있습니다.
